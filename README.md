# Design and Analysis of Algorithms

This repository contains my laboratory programs and practice implementations for the **Design and Analysis of Algorithms (DAA)** course.

- Merge Sort
- Quick Sort
- Bubble Sort
- Insertion Sort
- Kruskal's Minimum Spanning Tree
- Knapsack Problem
- External Sorting
- Random Number Generator
- Assembly Line Scheduling
- Algorithm Analysis Programs
- Other DAA practicals and assignments
